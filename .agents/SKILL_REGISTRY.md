## Cómo Usar Estas Skills

Las skills se cargan automáticamente cuando el contexto lo requiere:

- **Diseño UI/UX** → `premium-ui-design` se activa automáticamente
- **Marketing/Copy** → `copywriting` se activa automáticamente  
- **SEO** → `seo-audit` se activa automáticamente
- **Accesibilidad** → `accessibility-compliance` se activa automáticamente
- **Diseño Web** → `web-design-guidelines` o `frontend-design` según necesidad

## Reglas de Uso

1. Antes de escribir código de diseño → consultar `premium-ui-design`
2. Antes de escribir texto marketing → consultar `copywriting`
3. Antes de modificar SEO → consultar `seo-audit`
4. Todo código debe ser accesible → `accessibility-compliance`
