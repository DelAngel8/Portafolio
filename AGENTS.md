# AGENTS.md - Reglas para Agentes IA

## Reglas del Proyecto

### Comisiones permitidas
- **NUNCA** hacer commit sin antes consultar al usuario
- **NUNCA** agregar "Co-Authored-By" o atribuciones de IA en commits
- Usar commits convencionales (feat!, fix!, chore!, etc.)

### Before writing code
- Revisar CLAUDE.md para entender convenciones del proyecto
- Verificar que el código siga patrones existentes

### Before ending session
- Realizar `npm run lint` o verificar con `npm run dev` que el código funcione
- Reportar al usuario cualquier error o warning
